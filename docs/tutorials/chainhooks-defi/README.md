# Building a DeFi Monitor with Chainhooks

Learn how to build a real-time DeFi monitoring application using Chainhooks to track swaps, liquidity events, and whale transactions on Stacks.

## What You'll Build

A complete DeFi monitoring dashboard that:
- Tracks DEX swaps in real-time
- Monitors liquidity pool events
- Alerts on large (whale) transactions
- Displays live analytics

## Prerequisites

- Node.js 18+
- Basic TypeScript knowledge
- Familiarity with Stacks blockchain
- Hiro Platform account (for Chainhooks API)

## Tech Stack

- **Backend**: Fastify + TypeScript
- **Frontend**: React + Vite
- **Real-time**: WebSockets
- **Data**: Chainhooks API
